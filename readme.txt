A plugin that allows you to store the Elementor UI back to the old colors

== Description ==

Ele UI Color Scheme Restoration is a WordPress plugin that helps you to restore the original color scheme of Elementor's user interface. 
If you're a long-time Elementor user, you may have noticed that the user interface has undergone some changes in recent updates, which may affect your productivity or 
simply make it less pleasant to work with.  With this plugin, you can easily restore the old color scheme that you were used to, without having to 
install an older version of Elementor.



== Frequently Asked Questions ==

=  How does the Ele UI Color Scheme Restoration plugin work?
The plugin allows you to restore the original color scheme of Elementor's user interface. Once you install and activate the plugin, you can choose the color scheme that you prefer from the plugin settings, and the plugin will automatically apply the changes to the Elementor user interface.

=  Will the plugin affect my existing Elementor content?
No, the plugin only changes the color scheme of the Elementor user interface, it doesn't affect your existing content or templates.

= Is the plugin compatible with the latest version of Elementor?
Yes, the plugin is compatible with the latest version of Elementor, and it's regularly updated to ensure compatibility with future updates.

=  Can I customize the color scheme to my liking?
No, the plugin doesn't offer customization options for the color scheme. However, it does offer multiple color scheme options to choose from, so you can select the one that suits your needs and preferences.

=  Will the plugin slow down my website?
No, the plugin is lightweight and doesn't add any extra load on your website, so it won't slow down your website.

=  What happens if I deactivate the plugin?
If you deactivate the plugin, the color scheme of the Elementor user interface will revert to the default colors.

=  Is the plugin free or paid?
The plugin is free and available on the WordPress plugin repository. However, the developer may offer paid add-ons or premium support for advanced users.


== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Ele UI Color Scheme Restoration` and click 'Install Now' or click on the `upload` link to upload `ele-ui-color-scheme-restoration.zip`
4. Click on `Activate plugin`
